How to use cycle

August 24, 2015

As we all know a cycle goes around and around. In Ruby, a cycle can be used to do just that as an 'Enumerable.'

An enumerable is another way you can go about looping your code. You can also sort and manipulate certain objects with enumerables.

When we use cycle, we repeat the array or whatever is being called, over and over again. It can go on forever but like any loop, we can and should make it stop before we crash our computers.

What makes enumerables stand out and different from loops is the simple line of coding we can write with it. It's shorter, straight to the point and easily readable.

For a cycle, if we input an array that equals, "hi", "hola", "bonjour" the output would be: "hi", "hola", "bonjour", "hi", "hola", "bonjour", "hi", "hola", "bonjour"... until, well forever!
