What is Rails

October 11, 2015

Rails, aka "Ruby On Rails," created by David Heinemeier Hansson, is an open source framework that uses the Ruby language. It acts as a guide and helps developers find the best way to do things. Part of that includes keeping the code DRY and convention over configuration, where less yet beneficial decisions is key. Rails allows developers to use their time efficiently and have access to easy collaboration while having good maintanance support.

Everyting in Rails is an object allowing for a smooth learning transition for developers of various different languages. As long as you know how to use variables, functions, methods, loops, etc., Rails is an easy framework to adapt to. For the Ruby developers out there, Rails is a gem.

Rails is used by small startups and big companies alike. Github, Airbnb, Goodreads and Living Social are among those companies. Rails is even considered to be fun because of the rapid process and the accomplishment of more tasks. If you want to try it out for yourself, check it out on codecademy!
