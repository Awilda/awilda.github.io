Enumerable#map

September 20, 2015


What is an Enumerable?

An enumerable is the code block where each element from a list, such as from an Array or Hash, is passed though. With an enumerable, one could write code that makes each individual element do something.

What is an Enumerator

An enumerator allows us to chain methods to objects. When no block (enumerator) is passed, the enumerator will return

Using Enumerable#map

When using enumerable#map, you don't have to worry about it being destructive because a new array is returned with the results from the code block. If we decide to call on the original array, the original array appears the same. Now, the enumerable#map allows up to make changes to each element in an array.

Putting it together

Here's a visual of an enumerable: {|x| x + 1}

Here's a visual of an enumerator: [1,2,3,4].map

Here's what it looks like put together: [1,2,3,4].map {|x| x +1}

What we are doing here is using the enumerable#map to go through an array. The "x" is each number that is being pass from the array. In our code block, each number will be added to one. Therefore, our new array will return:

=> [2,3,4,5]
